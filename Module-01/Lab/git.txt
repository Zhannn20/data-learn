Обновление репозитория:

1. сделать индексацию. проверить статус git status

2. Добавить все файлы:  git add . 

3. проверим снова статус:  git status

4. Добавляем коммит : git commit -m "Сообщение"

5. снова проверяем статус:  git status 

6. отправляем файлы в репозиторий : git push



Создать новую ветку:

1. пишем : git checkout -b (название ветки, например  Test-learn)

2. Посмотреть в какой ветки мы находимся: git branch

3. чтобы перключится на другую ветку : git branch ( навзвание ветки например, master)
